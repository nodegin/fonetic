# fonetic
fonetic is an english-word like id generator

original idea: [see here](https://github.com/seejohnrun/haste-server/blob/master/lib/key_generators/phonetic.js)

# usage
```php
$generator = new Fonetic();
echo $generator->generate(6);
```
